# MitoRiboPy Release Notes: v0.1.0

Date: 2026-03-30

## Summary
This was the first package-oriented public release of MitoRiboPy. It established
the installable package layout, the `mitoribopy` CLI, and the initial CI and
test coverage needed for public use.

## Highlights
- Added the first package-oriented `src/mitoribopy` layout and CLI entrypoint.
- Added GitHub Actions CI to run tests on push/PR to `main`.
- Added tiny-fixture integration coverage for:
  - P-site/A-site offset selection behavior.
  - End-to-end CLI smoke outputs.
  - Codon usage output generation validation.
- Added package metadata polish in `pyproject.toml`.
- Added and standardized project changelog.

## Notes

- This version still included legacy compatibility layers while the package
  migration was being completed.
- Later releases replaced hardcoded reference data with packaged JSON/CSV files
  and removed the runtime dependency on `_legacy`.
